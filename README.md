# Image Search

![Image Search](./preview.png)

Image Search is a lightweight image search web application powered by the Pixabay API. It fetches images via Axios using async/await, displays results in a gallery (15 images per request), supports pagination via a “Load more” button, and provides a smooth viewing experience with SimpleLightbox and toast notifications via iziToast. Built with Vite.

## Live Application
- https://valentyn-m.github.io/goit-js-hw-12/

## Key Features
- Search images by keyword using the Pixabay API (paginated requests)
- Async data fetching with Axios (async/await)
- Results gallery with 15 images per request and “Load more” pagination
- Lightbox view with slideshow navigation using SimpleLightbox
- User notifications via iziToast (empty query, no results, last page, errors)
- Bundled and deployed with Vite
